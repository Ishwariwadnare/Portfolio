# Portfolio Website

A simple personal portfolio website scaffolded with React, Vite, and Tailwind CSS.

## Project structure

- `public/` - Static assets such as icons and resume files.
- `src/` - React application source code.
  - `assets/` - Media used by components.
  - `components/` - Reusable UI pieces.
  - `data/` - Static content and project definitions.
  - `pages/` - Page-level components.
- `tailwind.config.js` - Tailwind CSS configuration.
- `postcss.config.js` - PostCSS plugins configuration.
- `package.json` - Dependencies and scripts.

## Scripts

- `npm install` - Install dependencies.
- `npm run dev` - Start the development server.
- `npm run build` - Build for production.
- `npm run preview` - Preview the production build.

## Notes

Customize the `src/data/constants.js` file to update projects, skills, and contact details.
